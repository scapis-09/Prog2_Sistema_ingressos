Sistema de Compra Ingressos // Tela Inicial

OBJETIVO

O objetivo desse projeto é aprender sobre o desenvolvimento web por meio de uma atividade aplicada. Portando desenvolvemos um sistema de combra e venda de ingresos, contando com telas para clientes como para vendedores.

FUNCIONABILIDADES

  • Cadastro de eventos
  • Gerenciamento de ingressos
  • Visualização de vendas
  • Relatórios de vendas
  • Integração com Firebase para persistência de dados
  • Interface para compra

O QUE FOI PRATICADO

  • HTML, CSS e JavaScript
  • Manipulação de DOM
  • Integração de Banco de Dados (Firebase)
  • Criação de interfaces responsivas
  • Gerenciamento de eventos
  • Validação de formulários
